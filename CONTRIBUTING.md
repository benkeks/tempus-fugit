# Commit-Message-Convention:
* Commit messages should be in English.
* Commit message briefly sums up what was done: Fill in the sentence "This commit will..."

# Workflow
1. Choose an issue from To-Do
2. Create a WIP branch te implement the feature
3. Implement fefature
4. Resolve merge conflicts with master (pull continuously to make this easier)
5. Remove WIP-Status and assign Merge Request Manager to review the code
6. Merge into master
